Mentee Map

Mentee Map is a web application designed to help faculty efficiently monitor mentee goals, performance, and academic progress. The platform presents mentee survey insights through a clean, intuitive interface, enabling mentors to quickly identify strengths, trends, and areas for improvement. It also provides streamlined access to exam scores with advanced filtering and search tools, allowing faculty to make data-driven decisions with ease.
